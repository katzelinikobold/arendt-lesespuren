Transkription

Die Praktische Anwendung findet das Modell in der Erfas-sung bzw. der Transkription. Das bedeutet, dass die in den Faksimiles auszumachenden Lesespuren in die Struktur des definierten Datenmodells überführt werden müssen. An dieser Stelle ist es notwendig das Modell auf eine bestimmte Technologie zu übertragen, welche zur Persistenz der Daten verwendet werden soll. Die hier verwendete Technologie ist das JSON-Format, welches bereits zur Korpusbildung verwendet wude. Die Umsetzung ist allerdings keineswegs an JSON gebunden, sondern könnte ebenso mithilfe von XML erfolgen, falls eine andere Technologie zu dem jeweiligen Forschungsvorhaben besser passt. Eine Beispieltranskription, an der die Datenstruktur nachvollzogen werden kann und anhand derer der Transkriptionsvorgang im Folgenden genauer beleuchtet wird, ist nachfolgend unten abgebildet.
